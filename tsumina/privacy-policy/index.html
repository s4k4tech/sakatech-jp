プライバシーポリシー

Tsumina

当社は、積みプラ管理アプリ「Tsumina」（以下「本サービス」）において、ユーザーの個人情報保護を重要視し、個人情報保護法その他関連法令を遵守し、適切に取り扱います。

1. 個人情報の定義

本プライバシーポリシーにおいて「個人情報」とは、個人情報保護法に定める個人情報を指し、生存する個人に関する情報であって、当該情報に含まれる氏名、メールアドレス等により特定の個人を識別できるもの（他の情報と容易に照合でき、それにより特定の個人を識別することができることとなるものを含む）を指します。

2. 収集する情報

2.1 ユーザーが提供する情報

  • アカウント情報：メールアドレス、表示名、プロフィール画像
  • 認証情報：Google アカウント、Apple ID の基本情報
  • 商品情報：登録した商品データ、画像、メモ、タグ等
  • お問い合わせ情報：お問い合わせ内容、連絡先情報

2.2 自動的に収集される情報

  • デバイス情報：デバイスの種類、OS バージョン、アプリバージョン
  • 利用情報：アプリの利用状況、操作ログ、エラー情報
  • 技術情報：IP アドレス、ブラウザ情報、言語設定
  • 分析情報：Firebase Analytics による匿名化された利用統計

2.3 第三者サービスから取得する情報

  • 商品データ：バーコードスキャンによる商品情報（楽天API等）
  • 認証情報：Google、Apple から提供される認証関連情報

3. 情報の利用目的

3.1 サービス提供のため

  • アカウントの作成・管理・認証
  • 商品データの保存・同期・管理
  • バーコードスキャン結果の提供
  • クラウドデータの同期・バックアップ
  • カスタマーサポート・お問い合わせ対応

3.2 サービス改善のため

  • アプリの機能改善・新機能開発
  • ユーザー体験の向上
  • バグ修正・パフォーマンス改善
  • 利用統計の分析

3.3 運営・管理のため

  • 利用規約違反の監視・対応
  • セキュリティ維持・不正利用防止
  • 法的義務の履行

4. 第三者サービスの利用

本サービスでは以下の第三者サービスを利用しており、これらのサービスがデータを収集する場合があります：

4.1 Google Services

  • Firebase Authentication：アカウント認証
  • Cloud Firestore：データベース・データ同期
  • Firebase Storage：画像・ファイル保存
  • Firebase Analytics：利用統計分析
  • Firebase Crashlytics：クラッシュ報告・エラー分析
  • Google Sign-In：Google アカウント認証
  • Google Mobile Ads：広告表示

詳細：Google プライバシーポリシー

4.2 Apple Services

  • Sign in with Apple：Apple ID 認証
  • App Store Connect：アプリ内課金

詳細：Apple プライバシーポリシー

4.3 商品情報サービス

  • 楽天API等：バーコードスキャンによる商品情報取得

5. 情報の共有・提供

5.1 第三者への提供

以下の場合を除き、個人情報を第三者に提供することはありません：

  • ユーザーの同意がある場合
  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要な場合
  • 公衆衛生の向上または児童の健全な育成の推進のために特に必要な場合

5.2 業務委託

サービス提供に必要な範囲で、個人情報の取り扱いを委託する場合があります。委託先には適切な監督を行います。

6. データの保存・管理

6.1 保存場所

  • 国内：日本国内のサーバー
  • 海外：Google（米国）、Apple（米国）のサーバー

6.2 保存期間

  • アカウント情報：アカウント削除まで
  • 商品データ：ユーザーによる削除まで
  • ログデータ：収集から最大2年間
  • 分析データ：匿名化された状態で無期限

6.3 セキュリティ

  • データの暗号化
  • アクセス制御
  • 定期的なセキュリティ監査
  • インシデント対応体制

7. ユーザーの権利

7.1 アクセス権

自身の個人情報の利用状況について確認を求めることができます。

7.2 訂正・削除権

個人情報の訂正・削除を求めることができます。

7.3 利用停止権

個人情報の利用停止を求めることができます。

7.4 データポータビリティ

一定の形式でデータの提供を求めることができます。

8. Cookie・類似技術

本サービスでは、以下の技術を使用する場合があります：

  • Firebase SDK：認証・分析のため
  • 広告 ID：広告配信最適化のため
  • アプリ内分析：利用状況分析のため

9. 子どもの個人情報

13歳未満の子どもの個人情報を意図的に収集することはありません。13歳未満の子どもの情報を収集していることが判明した場合、速やかに削除いたします。

10. プライバシーポリシーの変更

本プライバシーポリシーを変更する場合、アプリ内通知またはその他適切な方法でお知らせします。重要な変更については、ユーザーの同意を求める場合があります。

11. お問い合わせ

個人情報の取り扱いに関するお問い合わせは、アプリ内のお問い合わせ機能をご利用ください。

12. 適用法令・管轄

本プライバシーポリシーは、日本法に準拠し、日本法に従って解釈されます。
